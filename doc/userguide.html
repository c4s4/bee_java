This file is the reference documentation for the Java Bee package. This package is made of tasks to build your Java projects and templates to quick start new ones.

Installation

Bee packages are managed as Ruby Gems, thus you can install Bee Java typing sudo gem install bee_java under Unix systems or gem install bee_java under Windows. Note that this will also install Bee if not already installed because Bee is a required dependency of bee_java.

Java Tasks

To get a reference about Java tasks, please see Tasks Reference section.

You can also get documentation about a given task in a terminal typing bee -k java.jar for instance. To list all available tasks in the Java package, type bee -k java.?.

Java Templates

To get a reference about Java templates, please see Templates Reference section.

You can also get documentation about a given template in a terminal typing bee -e java.lib for instance. To list all available templates in the Java package, type bee -e java.?.

Tools

This package comes with tools to convert dependencies file from and to maven project files.

gendeps

This tools generates a Bee dependencies file for existing Maven 2 project. To do so, type following command line in the root directory of the project:

gendeps > dependencies.yml

To generate this file, the tool runs Maven to get the list of dependencies using the command mvn dependency:list and parse result. The scope of Maven dependencies is converted into Bee scope with the following table:

Maven ScopeBee Scopes
compilecompile, test, runtime
providedcompile, test
runtimetest, runtime
testtest

Dependencies are searched recursively, thus generated list might be quite big as it includes all dependencies that will be included in CLASSPATH. You should check for collisions in this list as this is a quite common issue in the case of recursive dependencies.

maven2bee

This script converts a Maven (1 or 2) POM file into a Bee dependencies file. Dependencies are not searched recursively, thus you should prefer using the tool gendeps for Maven 2 projects. Maven scope is converted with the same table than with the gendeps tool.

To convert a Maven 1 project file you would type:

maven2bee project.xml > dependencies.yml

bee2maven

This script converts a Bee dependencies file into a Maven POM file (for the part related to dependencies). Note that it doesn't manage scopes and that you should manage them by hand in generated file.

To convert a Bee dependencies file into a Maven POM, type:

bee2maven dependencies.yml > project.xml

Enjoy!
